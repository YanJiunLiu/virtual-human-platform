[intro] 
    此專案組成為vit + react(typescript) + tailwindcss ，透過nodejs執行Server端，在由docker部屬
    主要執行相關邏輯則在src資料夾中

[安裝專案]
    首先拿到這包專案檔前，先安裝本機nodejs 
    https://nodejs.org/en/download

    安裝nodejs完成之後，在此資料夾執行npm install 安裝package.json中的套件

    以上安裝完成後可執行 npm run dev 進行本機網頁測試

    ● npm run build:user/build:admin 則會發布 html+js+css至dist資料夾中

    ● API相關的網址可由 src/js/config.js做修改


[nodejs環境測試]
    後端server是使用nodejs 
    指令 node server.js 則會啟動後端執行網頁
